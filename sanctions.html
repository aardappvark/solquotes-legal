3. Geographic Screening

Sol Quotes performs on-device geographic screening at launch using the following signals (in priority order):

  1. SIM Operator Country Code — mobile country code from the device’s SIM
  2. Network Operator Country Code — country of the currently connected network
  3. Device Locale — country/region setting on the device
  4. Geocoder Reverse Lookup — if coarse location permission is granted

Note: If any signal indicates a restricted jurisdiction, access will be blocked. A match on any single signal is sufficient.

4. Privacy-Preserving Design

Your privacy is protected. All geographic screening is performed entirely on-device. No location data or country codes are ever transmitted to any external server.

5. User Representations

By using Sol Quotes, you represent and warrant that you are not located in, ordinarily resident in, or a citizen or national of any restricted jurisdiction listed above, and that you are not on any applicable sanctions list.

6. Circumvention Prohibited

You must not use VPNs, proxies, or other technical means to circumvent Sol Quotes’ geographic restrictions. Doing so constitutes a violation of these terms and may result in termination of access.

7. Updates

This list may be updated at any time to reflect changes in international sanctions regimes. Continued use of Sol Quotes constitutes acceptance of the current restrictions.
